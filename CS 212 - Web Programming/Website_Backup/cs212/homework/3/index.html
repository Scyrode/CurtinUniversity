Custom Fonts In Your Webpage Using Google Webfonts

There are multiple ways to use custom webfonts in your website. The method that I prefer most is using Google Webfonts. Google has provided a set of professional fonts for developers to use that are completely free of charge. In order to use a Google Webfont in your HTML website, follow the steps below:

  1. Navigate to https://fonts.google.com/ and find a font that you like.
  2. Select the weight of the font that you would like to use in your website by clicking on "+ select this style"
  3. From the side menu under "Use on the web", copy the <link> tag and paste it into the head of your html document
  4. In you CSS file, modify the default "font-family" of a specific tag using: "font-family: 'Bungee Shade', cursive;"
