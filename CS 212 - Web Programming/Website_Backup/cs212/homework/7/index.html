Addendum


                
Problem Description


                
The problem that this webpage solves is finding the min, max and average of a list of numbers. The user inputs any 5 numbers that they would like the calculation to be performed on and then click "start calculation". The webpage then find the goes through the number list and prints the min and max of the numbers inputted. It also calculates the average value from the numbers inputted and prints it as well.


                
Challenges


                
"null has been blocked by CORS policy" JS error


                
Despite connecting up the javascript file correctly to the html page using the 'script' tag, the javascript did not work, showing the error: "null has been blocked by CORS policy". It turns out that in order for the js script to connect properly to the html file, the html page cannot be opened using a file explorer but instead must be opened through a host. So after uploading the html along with the css and js files to my website, the error went away and the javascript worked


                
Fitting the iamge ad inside of the 'aside' section


                
After placing the image inside of the 'aside' section, it did not resize to fit the 'aside' section. In order to fix that, its width property was set to 100%, to occupy all the available width of its parent container (i.e: 'aside' section) and the height was set to 'auto' to maintain the photo's aspect ratio.
